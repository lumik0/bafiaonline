# Bafia Online

Бафия онлайн - это неофициальная браузерная версия игры «Мафия Онлайн», которая добавляет в игру дополнительный функционал и модификации

в игру можно играть как с компьютера, так и с телефона(айфон должен поддерживаться)

## Запуск

Онлайн: [тут](https://lumik0.github.io/bafiaonline/run/index.html)

Локально: Скачайте архив, распакуйте его, откройте `index.html` из папки `run` и наслаждайтесь игрой

## для тех, кто хочет делать моды или собственную версию
необходимо установить [bun](https://bun.sh).

после установки откройте терминал в папке проекта и выполните:
```bash
bun cb
```
