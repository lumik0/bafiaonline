# Bafia Online
Mafia Online Browser Version

Это неофициальная браузерная версия официальной игры Мафия Онлайн

В ней есть всякие приколы

в эту можно играть с компьютера так и с телефона(с айфона можно)

скачиваешь архив, разархивируешь его, запускаешь index.html из папки run и наслаждайтесь игрой

## для тех кто хочет сделать моды или свою собственную версию:
вам нужно установить [bun](https://bun.sh)

открываешь терминал прописываешь `bun cb`

скоро..
